### 2. 단축키

- **`S`** 객체 크기 변경  
- **`G`** 객체 이동  
- **`Tab`** 객체 편집모드  
- **`Alt + Z`** 객체 투명화 
- **`Ctrl + R`** 객체 분할   
- **`Ctrl + Shift + B`** 포인트 크기 변경  
- **`Shift + D`** 객체 복제  
- **`Shift + D + Mouse Wheel`** 수평맞추면서 복제
